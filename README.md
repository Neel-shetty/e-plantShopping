# e-plantShopping

## Paradise Nursery Shopping Application

A React-based shopping cart application for an online plant shop offering a variety of house plants.

### Features

- **Landing Page**: Welcome page with company information and "Get Started" button
- **Product Listing**: Browse plants organized by categories (Air Purifying, Aromatic, Insect Repellent, Medicinal, Low Maintenance)
- **Shopping Cart**: Add/remove plants, adjust quantities, view totals
- **Redux State Management**: Centralized cart state management using Redux Toolkit

### Technologies Used

- React 18
- Redux Toolkit
- Vite
- CSS3

### Getting Started

1. Clone the repository
2. Run `npm install` to install dependencies
3. Run `npm run dev` to start the development server
4. Open your browser and navigate to the provided URL
